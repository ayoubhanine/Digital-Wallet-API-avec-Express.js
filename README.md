Creation de serveur(serveur.js) 
1-On importe Express et les routes.
2-On crée l’application et configure le port.
3-On ajoute le middleware pour parser le JSON.
4-On définit nos routes principales (/users et /wallets).
5-On gère les routes inexistantes avec un 404.
6-On démarre le serveur pour écouter les requêtes.
